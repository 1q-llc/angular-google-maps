The map-control directive is used to add custom controls to the maps UI.

Usage

Parameters

Required properties are in red.

Param Type Details
template expression Template to be used for the control's content.
position string String representing the position of the control within the map UI. Should be of the form "top-left" or "TOP_LEFT". Defaults to "TOP_CENTER". See Control Positioning for position options and behaviour.
controller expression|function Controller applied to the control's template.
index expression

Index applied to the map control. Controls are placed at each location by the order of the index; controls with a lower index are placed first.

By default, all custom controls are placed after any API default controls but this can be overridden using a negative index. Custom contols cannot be placed to the left of the logo or the right of the copyrights.
