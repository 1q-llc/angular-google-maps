New in v1.2.0

The ui-gmap-free-draw-polygons directive is used to draw polylines which are converted to polygons. Eventually this may be extended to draw polygons and shapes. The attributes options and events are planned to be added as well for styling and polygon callbacks. If you need mouse click events then grab the events from the google-map directive.

Usage

Parameters

Required properties are in red.

Param Type Details
polygons expression Polygon array to be bound too. It is bidirectional. However, its main intent is to create polygons from the directive to the controller. But if the polygons are modified on the controller side, the changes should be reflected to the directive as well. If your getting crazy with polygon requirements then use the standard polygon, polygons, polyline or polylines directives.
draw expression (undefined function) Originally this should be undefined on the controller side. The scope.draw is assigned on the directive side. This allows the controller to directly call the directive and call the draw function to engage the directive to initiate drawing mode.
