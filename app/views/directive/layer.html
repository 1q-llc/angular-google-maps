The layer directive is used to draw different types of layers on the map.

Usage

<layer 
	type='{string}'
	namespace='{string}'
	options='{expression}'
	show='{expression}'>
	...
</layer>

Parameters

Required properties are in red.

Param Type Details
type string The type name of the layer. Available layer types
namespace string Some layers are part of a different namespace. For example, the WeatherLayer is part of the weather namespace. Use this attribute to specify the correct namespace of the layer you want to use.
options expression Object containing the options for the requested layer type. See the different layer types for information about their respective options.
show expression boolean value which shows or hides the layer
