The polygon directive is used to draw polygon shape layers on the map. See Polygon API.

Usage

<polygon path='{expression}'
	fill='{expression}'
	stroke='{expression}'
	clickable='{expression}'
	draggable='{expression}'
	editable='{string}'
	geodesic='{expression}'
	icons='{expression}'
	visible='{expression}'>
	...
</polygon>

Parameters

Required properties are in red.

Param Type Details
path expression An array of objects containing latitude and longitude properties which define the points connected by the polyline
fill expression Object to configure the polygon fill type. Valid properties are color (CSS hex notation) and opacity (between 0.0 and 1.0)
stroke expression Object to configure the line stroke. Valid properties are color (CSS hex notation), weight (int) and opacity (between 0.0 and 1.0)
clickable expression boolean property to tell the polygon to respond to click events
draggable expression boolean property to make the polygon draggable across the map
editable expression boolean property to make the polyline editable by dragging control points
geodesic expression boolean property to make the polyline follow the curvature of the earth
icons expression url to the icon image to use for each marker
visible expression boolean property which shows or hides the polygon
