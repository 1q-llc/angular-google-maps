The google-map directive is the core of all other directives. It is used to instantiate a Google Map object in your page. Every other directive must be contained within a google-map for them to work.

Usage

<google-map center='{expression}' 
	zoom='{expression}'
	draggable='{string}'
	dragging='{expression}'
	refresh='{expression}'
	options='{expression}'
	events='{expression}'
	bounds='{expression'}}>

	...

</google-map>

Parameters

Required properties are in red.

Param Type Details
center expression Expression to evaluate as an object representing the map center. The object must have latitude and longitude numeric properties.
zoom expression Expression to evaluate as the map's zoom level. This must be a number between 1[1] and 20[2].
draggable string true if map must be draggable, false otherwise
refresh expression Expression which, if it evaluates to true, will trigger a map refresh.
options expression Additional options to pass to the google.maps.Map constructor
events expression Custom events to apply to the map. This is an associative array, where keys are event names and values are handler functions.
bounds expression
