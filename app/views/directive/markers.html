The markers directive is similar to the marker diective above, but is used when a lot of markers need to be added to a map.

Usage

<markers models='{expression}'
	coords='{string}'
	labelContent='{string}'
	icon='{expression}'
	click='{expression}'
	options='{expression}'>

	...

</markers>

Parameters

Required properties are in red.

Param Type Details
models expression
icon expression
