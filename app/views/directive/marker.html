The marker directive is used to add a google.maps.Marker object to an existing map.

Usage

Parameters

Required properties are in red.

Param Type Details
coords expression Object having latitude and longitude properties representing the position of this marker
icon expression Epxression returning the absolute path to an image used as the marker icon
click expression|function Event handler to call when clicking on the marker
options expression Object containing additional options for the marker. See MarkerOptions.
events events Object containing event handlers for this marker. The keys are the event names and the values are the handler functions. See Marker documentation (under Events section) for a list of all events.
