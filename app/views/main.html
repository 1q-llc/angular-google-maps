Google Maps for AngularJS

AngularJS directives to integrate Google Maps into your applications.

Download (v1.0.4) View on GitHub

About

Google Maps for AngularJs is a set of directives written in CoffeeScript which integrate Google Maps in AngularJS applications.

It is based on the Google Maps Javascript API version 3.

Features

There are directives for most of the widely-used Google Maps objects, including markers, windows, lines and shapes. We're currently working on integrating more layer objects, including the HeatMap and such.

Contributing

The code is hosted on GitHub. If you need a feature which isn't implemented yet, please use our issue tracker to open a feature request, or fork our repository, implement the feature and issue a pull request. Contributions are more than welcome!

We use the git flow branching model. So if you are interested in contributing, we encourage you to learn about it.
