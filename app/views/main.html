All new API!

The complete angular-google-maps code has been overhauled to simplify its use and to make the development process more straightforward. Directives have been split in separate source files and test specs have been added to the project (thanks to nmcready).

This results in an easier integration in your project, a better API documentation and much more simplified bug hunting.

Features

angular-google-maps offers directives for the following Google Maps elements:

  • Map
  • Marker
  • Label
  • Polygon
  • Polyline
  • TrafficLayer
  • InfoWindow

If you need a feature which isn't implemented yet, please use our issue tracker to open a feature request, or fork our repository and issue a pull request!
