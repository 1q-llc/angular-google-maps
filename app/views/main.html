Google Maps for AngularJS

This is a set of directives to integrate Google Maps™ into your AngularJS applications. It works with both 1.0.x and 1.2.x versions of AngularJS.

Features

There are directives for most of the widely-used Google Maps objects, including markers, windows, lines and shapes. We're currently working on integrating more layer objects, including the HeatMap and such.

Contributing

If you need a feature which isn't implemented yet, please use our issue tracker to open a feature request, or fork our repository, implement the feature and issue a pull request. Contributions are more than welcome!
