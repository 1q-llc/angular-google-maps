GoogleMapsApi Provider New in v2.0.0

Alias: (Google Maps SDK Async Loader)

This provider is was created for a few reasons:
  • google recommends loading the sdk this way
  • It garuntees that angular-google-maps does not begin processing any directives until all of the google maps sdk is fully ready!
To utilize this provider you must load it and configure it properly. Please note the defaults!

Configure

//'GoogleMapApiProvider'.ns() == 'uiGmapGoogleMapApiProvider' .config(['GoogleMapApiProvider'.ns(), function (GoogleMapApi) { GoogleMapApi.configure({ // key: 'your api key', v: '3.17', libraries: 'weather,geometry,visualization' }); }])

When is Google SDK ready?
