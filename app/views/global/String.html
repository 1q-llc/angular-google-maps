String Prototypes

Following prototypes were added:
  • ns(): alias to flare
  • flare(): Prepends 'uiGmap' to any string. This makes changing the global namespace for the entire application easy.
    contains(stringToSearch): Returns true if a stringToSearch exists in this string.
source code
var someStr = "some".ns(); console.info(someStr); //out puts "uiGmapsome" var doesHaveUiGmap someStr.contains("uiGmap"); //doesHaveUiGmap is now true
