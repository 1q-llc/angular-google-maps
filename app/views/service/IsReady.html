IsReady Service

This service lets you know when ui-gmap is ready. As IsReady passes a promise callback of maps. Beyond that; the instances passed back to you come back with the following info:
  • instance: map instance number (internal counter for ui-gmap on which map)
  • map: The actual gMap (google map sdk instance of the google.maps.Map).
    map._id: A unique UUID that gets assigned to the gMap via ui-gamp api.
source code
.controller("someController",['$scope', 'Logger'.ns(), function ($scope, Logger) { IsReady.promise(2).then(function (instances) { instances.forEach(function(inst){ var map = int.map; var uuid = map._id; var mapInstanceNumber = inst.instance; //starts at 1 }); }); });
