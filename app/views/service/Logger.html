Logger Service

This logger utilizes $log underneath. However unless you pull Logger into your contoller you will not see any error or info logs from the ui-gmap api. So if you want to see logs set Logger.doLog = true. (defaults to false!)
source code
//TODO in the future add log levels // 'Logger' = 'uiGmapLogger' .controller("someController",['$scope', 'Logger'.ns(), function ($scope, Logger) { Logger.doLog = true; log.info("hmm"); log.debug("huh!"); log.warn("uhoh!"); log.error("oh crap!"); });
