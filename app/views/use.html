Quickstart

To start using Google Maps for AngularJS, follow these simple steps to get started. Once you're up and running, visit the API documentation to learn how to get the most out of it.

  1. Download angular-google-maps.js (minified version) and put it with your other scripts. Alternatively, you can use bower to install it automatically:
    bower install angular-google-maps
    If you're not using bower (YOU SHOULD), you'll need to fetch:
    • jquery.js
    • lodash.js
    • bluebird.js use a real version! New in v2.0.0
    and save them it alongside your other scripts.
  2. Load/include lodash, bluebird and angular-google.maps.js
  3. Include the Google Maps API v3, via one of two ways:
    • New in v2.0.0
      GoogleMapsApi Loader (Async)
      This is the RECOMMENDED way as it is trying to garuntee that google maps is ready prior to running any google dependencies!
    • Load into your html page:
      Warning
      Note: If you go this route this must be loaded prior to angular-google-maps.js
    Serving Google Maps in China (mainland)

    It is a state demand that all online map providers must use an obscured coordinate system called GCJ-02 (aka Coordinates on Mars). GCJ-02 is WSG-84 based, but added offsets to both latitude and longitude.

    If you display a marker from GCJ-02 coordinates on a GCJ-02 map, the place will be marked correctly.
    However the offsets can result in a less-than-100 up to 700 meter error from the actual location if you place a GCJ-02 marker on a WSG-84 map and vice versa.

    There is an open-source project that can provide approximate translation between GCJ-02 and WSG-84.

    Google also submits to this regulation. They serve this modified system at maps.google.cn.
    Specify a china flag when you load the Google Maps API using the GoogleMapsApi Loader,
    or replace //maps.googleapis.com/maps/api/js with http://maps.google.cn/maps/api/js if you manually load Google Maps API via a script tag.

  4. Make your application module depend on the google-maps module:
    //note ns() (global string prototype), //this prepends uiGmap to 'google-maps', so it is really 'uiGmapgoogle-maps' angular.module('myApplicationModule', ['google-maps'.ns()]);
  5. Add a map object in your scope like this:
    $scope.map = { center: { latitude: 45, longitude: -73 }, zoom: 8 };

    Note: this is the very minimal set of properties required for the map to work. See the API documentation for the full list of supported properties.
