Quickstart

To start using Google Maps for AngularJS, follow these simple steps to get started. Once you're up and running, visit the API documentation to learn how to get the most out of it.

  1. Download angular-google-maps.js (minified version) and put it with your other scripts. Alternatively, you can use bower to install it automatically: 
    bower install angular-google-maps
  2. Include the Google Maps API v3 script and angular-google.maps.js in your index.html or layout template: 
    <script src='//maps.googleapis.com/maps/api/js?sensor=false'></script>
<script src='/path/to/angular-google-maps[.min].js'></script>
  3. Make your application module depend on the google-maps module: 
    angular.module('myApplicationModule', ['google-maps']);
  4. Add a map object in your scope like this: 
    $scope.map = {
  center: {
    latitude: 45,
    longitude: -73
  },
  zoom: 8
};

    Note: this is the very minimal set of properties required for the map to work. See the API documentation for the full list of supported properties.

  5. Add a <google-map> element in your template like this: 
    <google-map center='map.center' zoom='map.zoom'></google-map>
  6. Specify an height via CSS for the map container: 
    .angular-google-maps-container { height: 400px; }
