Logger Service

This logger utilizes $log underneath. However, unless you pull uiGmapLogger into your contoller, you will not see any error or info logs from the ui-gmap API. So if you want to see logs, set uiGmapLogger.doLog = true (defaults to false).
source code
.controller("someController", function(uiGmapLogger) { uiGmapLogger.doLog = true; log.info("hmm"); log.debug("huh!"); log.warn("uhoh!"); log.error("oh crap!"); });
