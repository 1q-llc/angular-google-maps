Lodash Extensions

Set of utility functions added to lodash / underscore.
source code
  • intersectionObjects:
    Intersection of Objects if the arrays have something in common each intersecting object will be returned. Arguments: array1, array2, comparison (optional) || defaults to _.isEqual
  • containsObject:
    Utilizes _.any underneath with an optional comparison function vs _.isEqual
    Args: obj, target (thing to look for), comparison (optional) || defaults to _.isEqual
  • differenceObjects:
    Filters all the differences between array1 and array2.
    Args: array1, array2, comparison (optional) || defaults to _.isEqual
  • withoutObjects: Alias to differenceObjects.
  • indexOfObject:
    Find the index of an item in an array via a comparison function.
    Args: array, item, comparison (required), isSorted (if true the performance is better)
  • extends:
    Iterates of an array of object to run _.extend on. Very similar to _.merge. arrayOfObjectsToCombine
  • isNullOrUndefined: Returns truthy if something is null or undefined.
