events expression

Custom events to apply to the {{thing}}. This is an associative array, where keys are event names and values are handler functions. See {{thing}} events.

The handler function takes four parameters (note the args are expanding on the original google sdk's default args):
  1. {{thing}}: the GoogleMaps {{thing}} object
  2. eventName: the name of the event
  3. model: internal angular-google-map model being used to generate the google {{thing}} object. This can be a model (plural directive) and or scope (singular directive).
  4. arguments: the arguments provided by Google Maps for this event type. See the {{thing}} docs
