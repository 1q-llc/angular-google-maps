The search-box directive is used to add a Google Places search box to the map.

Usage

Parameters

Required properties are in red.

Param Type Details
template expression Template to be used for the search box's input field.
position string String representing the position of the control within the map UI. Should be of the form "top-left" or "TOP_LEFT". Defaults to "TOP_LEFT". See Control Positioning for position options and behaviour.
parentdiv expression The id of another div on the page to which the the Search Box field should be appended. If this property is set the Search Box will not be added as a control on the map and the position parameter is not required.
events expression

Object containing event handlers for this Search Box. The keys are the event names and the values are the handler functions. See SearchBox documentation (under Events section). There is only one event, places_changed.
